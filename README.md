# randomvpn

a simple script for connecting to publicly available vpns

## requirements

- openvpn
- python3

## macOS/linux

```git clone https://github.com/flinnthebin/vpn```   
```chmod 755 randomvpn```   
```sudo ./randomvpn```   

## windows

download openvpn - https://openvpn.net/client-connect-vpn-for-windows/
run powershell as administrator

```git clone https://github.com/flinnthebin/vpn```   
```python3 randomvpn```
