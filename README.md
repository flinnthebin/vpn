# vpn

a simple script for connecting to publicly available vpns

## requirements

- openvpn
- python3

## linux usage

```git clone https://github.com/flinnthebin/vpn\n```
```chmod 755 randomvpn```
```sudo ./randomvpn```

## windows usage

- download openvpn
- run powershell as administrator
- ```python3 randomvpn```
